# Gene-Expression-Profiling-of-Cardiogenic-Shock-Patients-under-ECMO
Multi-omics analysis of ECMO-treated cardiogenic shock patients. Includes R code, report &amp; slides. Combines ML (Random Forest, LDA, LASSO, SCUDO) and network-based functional integration (g:Profiler, pathfindR, rCCA) to identify transcriptomic predictive signatures and systemic drivers of clinical outcomes.
